# 전체 프로세스

## 데이터 처리 (문서 기준 설명)

> 아마도 가장 중요한 부분이지 않을까 싶다.

- 문서를 읽어온다.
- 청킹을 한다. (Splitter 이용)
- Embedding을 이용해서 벡터로 변환한다.
- 전환된 벡터를 DB에 저장한다.

## 질의 처리

- 데이터를 Retriver로 읽어온다.
- 가져온 데이터 중에 유사도를 확인해서 답변을 선택한다.
- 참고해서 답변을 구성한다.
